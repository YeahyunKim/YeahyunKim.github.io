## 01 문제 :덧셈식 출력하기

---
>#### 문제 설명

>두 정수 `a`, `b`가 주어질 때 다음과 같은 형태의 계산식을 출력하는 코드를 작성해 보세요.
{: .command-text}

```python
a + b = c
```
<BR>
<BR>
>#### 제한사항

>• 1 ≤ `a`, `b` ≤ 100
{: .command-text}
<BR>
<BR>

>#### 입출력 예

##### 입력 #1
```python
4 5
```

##### 출력 #1
```python
4 + 5 = 9
```
<BR>
<br>
>#### 제시 코드

```python
a, b = map(int, input().strip().split(' '))
```

<br>
<br>
<BR>
## 02 풀이 
---

>#### 1) 내가 푼 방법 - f-string 활용하기

>Python 3.6 이상에서는 `f-string`을 사용하여 문자열 안에 변수를 직접 삽입할 수 있다.  
>예를 들어, a = 4일 때, `print(f"a = {a}")` 와 같이 사용할 수 있다.  
>**이 방법이 가독성이 더 좋아, 자주 활용할 것 같다.**
{: .command-text}

```python
print(f"{a} + {b} = {a+b}")

# 4 + 5 = 9
```

<br>

>#### 2) 내가 푼 방법 - 연산식 사용하기

>`a`와 `b` 를 문자열로 형변환 하고, 연산식을 활용하여 문제를 풀 수 있다.
{: .command-text}

```python
print(str(a) + " + " + str(b) + " = " + str(a+b))

# AbCdEfG
```

<br>
<br>
<br>

<BR>
## 03 Answer
---
>#### 1) 내가 푼 방법 - f-string 활용하기

```python
a, b = map(int, input().strip().split(' '))

print(f"{a} + {b} = {a+b}")

# 4 + 5 = 9
```

<br>


>#### 2) 내가 푼 방법 - 연산식 사용하기

```python
a, b = map(int, input().strip().split(' '))

print(str(a) + " + " + str(b) + " = " + str(a+b))

# 4 + 5 = 9
```
