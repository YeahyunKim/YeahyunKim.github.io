## 01 문제 :문자열 출력하기

---
>#### 문제 설명

>문자열 `str`이 주어질 때, `str`을 출력하는 코드를 작성해 보세요.
{: .command-text}
<BR>
<BR>
>#### 제한사항

>• 1 ≤ `str`의 길이 ≤ 1,000,000  
>• `str`에는 공백이 없으며, 첫째 줄에 한 줄로만 주어집니다.
{: .command-text}
<BR>
<BR>

>#### 입출력 예

##### 입력 #1
```python
HelloWorld!
```

##### 출력 #1
```python
HelloWorld!
```
<BR>
<br>
>#### 제시 코드

```python
str = input()
```

<br>
<br>
<BR>
## 02 풀이 
---

>원하는 값을 출력할 수 있는 방법은 `print(값)`을 활용하여 출력할 수 있다.
{: .command-text}

```python
print(str)
```

<br>
<BR>
<br>
## 03 Answer
---

```python
str = input()

print(str)
```
