## 01 문자열 출력하기


```
import Foundation

let s1 = readLine()!
```

제시된 내용을 보면,
현재 변수 `s1` 에는 "HelloWorld" 라는 `string` 형태의 값이 들어가 있다.
따라서 해당 String 값을 출력하기 위해서는

`print(s1)` 을 입력하면 된다.

---

## Answer

```
import Foundation

let s1 = readLine()!

print(s1)
```
